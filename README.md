Python exercises

## Exercise 01
Given a dictionary:
```
D = {'a': 1, 'c': 3, 'b': 2}
```
Print the record list by rearranging the keys:
```
a => 1
b => 2
c => 3
```

## Exercise 02

```
spam pams amsp mspa
```

The output should be:
